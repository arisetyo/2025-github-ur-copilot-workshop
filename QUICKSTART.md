# 🚀 Quick Start - Simplified Pomodoro Timer

Get started in under 1 minute!

## One-Command Install & Run

```bash
# Install Flask and run
pip install flask && python simple_app.py
```

That's it! Open http://localhost:5000 in your browser.

## What You Get

✅ Full Pomodoro timer functionality
✅ 25-minute work sessions  
✅ 5-minute short breaks
✅ 15-minute long breaks (every 4 sessions)
✅ Clean, responsive UI
✅ Browser notifications

## For Production Use

```bash
# Install in virtual environment
python3 -m venv .venv
source .venv/bin/activate
pip install flask

# Run (debug disabled by default for security)
python simple_app.py
```
